About Me

Full Stack Developer

I am an Android and iOS Developer skilled in creating modern, high-performance mobile applications with seamless RESTful and GraphQL API integration.

My expertise includes Kotlin, Swift, and Kotlin Multiplatform (KMP), allowing me to build both native and cross-platform apps efficiently. I focus on writing clean, maintainable, and scalable code using modern architecture patterns such as MVVM, Repository, and Dependency Injection (Hilt, Dagger, or Koin).

I specialize in connecting mobile applications with secure and efficient backend systems using Retrofit, OkHttp, and Apollo GraphQL. My experience includes implementing authentication mechanisms like OAuth2 and JWT, ensuring data privacy and robust communication between client and server.
